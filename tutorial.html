Anleitung zur Bedienung:

1. Nummern speichern

  1. Klicken Sie auf "Tutorial anzeigen", um diese Anleitung zu öffnen.
  2. Geben Sie in die Felder "Nummer 1" und "Nummer 2" die gewünschten Zahlen ein.
  3. Alternativ können Sie die Nummern scannen:
    • Klicken Sie auf "Barcode scannen"
    • Wählen Sie aus, ob Sie "Nummer 1" oder "Nummer 2" scannen möchten
    • Richten Sie die Kamera auf den Barcode
  4. Klicken Sie auf "Speichern", um die Kombination zu speichern.
  5. Die Kombination wird in der Liste unten angezeigt.

2. Nummern löschen

  1. Geben Sie die Nummer ein, die Sie löschen möchten, in das Feld "Nummer zum Löschen".
  2. Alternativ können Sie die Nummer scannen:
    • Klicken Sie auf "Barcode scannen"
    • Wählen Sie "Nummer zum Löschen" aus
    • Richten Sie die Kamera auf den Barcode
  3. Klicken Sie auf "Löschen", um die Kombination zu entfernen.
  4. Die Seite wird automatisch aktualisiert, um die geänderte Liste anzuzeigen.

3. Kombinationen anzeigen

  1. Klicken Sie auf "Laden", um alle gespeicherten Kombinationen anzuzeigen.
  2. Die Kombinationen werden in einer Liste unterhalb der Buttons angezeigt.
  3. Klicken Sie auf "Kopieren", um die Liste in die Zwischenablage zu kopieren.
